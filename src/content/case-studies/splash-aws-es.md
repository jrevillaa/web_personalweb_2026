## El Reto
Tener una plataforma Moodle en cloud implementada desde cero, asegurando configuración correcta y lista para crecer según demanda.

## La Solución
Implementación en **AWS**:
* Aprovisionamiento de servidor y dependencias.
* Instalación y configuración de Moodle (stack web + base de datos).
* Configuración inicial de seguridad y buenas prácticas (cron, backups básicos, caches).
* Validación funcional de acceso y operación base.

## Resultados
* Moodle desplegado en **AWS** y listo para puesta en marcha.
* Base escalable para crecer en cursos/usuarios.
* Configuración inicial alineada a operación estable.
