## El Reto
Contar con una app móvil institucional con identidad propia, lista para distribución, manteniendo compatibilidad con el backend Moodle y una experiencia consistente para el alumno.

## La Solución
Implementamos una **App Moodle** sobre base 4.3 con enfoque “ready-to-publish”:
* Branding completo (logos, colores, textos, splash, app icons).
* Configuración de endpoints y políticas de sesión/notificaciones.
* Ajustes de UX para navegación y acceso a contenidos clave.
* Preparación de builds y checklist para publicación (metadatos, versionado, firmas).

## Resultados
* App institucional lista para **Play Store** y **App Store**.
* Experiencia móvil consistente con el ecosistema Moodle.
* Reducción de fricción de acceso para estudiantes (menos pasos, más adopción).
