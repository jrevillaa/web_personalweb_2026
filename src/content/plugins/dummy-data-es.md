## Descripción General

**Dummy Data Generator** acelera QA y pruebas de carga: crea un entorno representativo sin esfuerzo manual.

### Buenas Prácticas

1. **Reproducibilidad**
   * Semillas configurables para datasets repetibles.
2. **Seguridad**
   * Genera datos ficticios, sin PII real.
3. **Escalabilidad**
   * Jobs por cron o CLI para crear grandes volúmenes sin timeouts.
4. **Limpieza**
   * Opciones de rollback/limpieza para iterar rápido.
