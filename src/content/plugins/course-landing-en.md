## General Description

**Course Landing Pages (SEO + Checkout)** transforms Moodle into a sellable system: each course can have its optimized public landing page and a purchase/enrolment flow.

### Technical Implementation

1. **Content Model**
   * Fields per course (price, duration, date, banner, benefits).
   * Templates rendered securely (no unsafe HTML).

2. **SEO & Sharing**
   * Metatags: title/description, canonical, robots.
   * OpenGraph/Twitter Cards for social, sharing.
   * Friendly URLs and optional sitemap (depending on configuration).

3. **Conversion**
   * Free course: registration/enrolment with 1–2 steps.
   * Paid course: integration with gateway and automatic enrolment.
   * Event tracking (view → checkout → payment) if analytics is integrated.
